Key Distinction

Legitimate IDN: Uses Unicode characters from a single script that are appropriate for the domain's language and region.

Detection Logic

PhishingURLAnalyzer should NOT flag IDNs as phishing if:

  • All characters come from the same Unicode script (e.g., all Cyrillic, all Arabic)
  • The domain doesn't visually resemble a popular Latin domain
  • No suspicious TLD or other phishing indicators
