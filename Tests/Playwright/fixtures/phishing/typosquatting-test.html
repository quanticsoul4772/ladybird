Typosquatting Techniques

Character Substitution: amazonn.com ‚Üí amazon.com Distance: 1

Detection Method

PhishingURLAnalyzer uses Levenshtein Distance (edit distance) to compare domain names against a database of 100+ popular domains.

  • Edit Distance 1-3: Flagged as typosquatting (25% phishing score)
  • Edit Distance 0: Exact match, legitimate
  • Edit Distance > 3: Too different, not typosquatting
